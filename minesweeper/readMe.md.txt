Game Rules
1. Left click on any block to reveal its value.
2. If the block is a Bomb then you lose and the game is over. 
3. Right click on a hidden block to flag it as a bomb. 
4. A number value indicates how many bombs are touching that specific block. 
5. An empty block will reveal its surrounding blocks. 
6. There is no time limit. 
7. Reveal all off the blocks and flag all of the bombs correctly to win.

How to install and use app
1. Open command line.
2. Navigate to folder called minesweeper(should contain public and src folders).
3. Type "npm install" in command line and wait for app to install.
4. Type "npm start" in command line. After a moment the browser should open with the game ready to play.
